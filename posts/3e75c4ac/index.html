node版本12.22, win10

执行npm install报错信息如下

  • 解决办法
  1. 以管理员运行cmd
  2. npm install --g --production windows-build-tools
  • 如果出现Successfully installed Python 2.7卡住了的情况，解决办法如下
  1. 在资源管理中搜索%temp%，window系统win+R搜索就可以
  2. 在搜索到的文件夹%temp%中创建dd_client_.log文件，并且填入内容Closing installer. Return code: 3010.
